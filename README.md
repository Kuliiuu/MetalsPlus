# MetalsPlus (Fabric 1.21.1)

MetalsPlus is a lightweight and expandable Fabric mod for Minecraft 1.21.1 that introduces new metals, materials, and later world generation features. The goal of the mod is to enhance mid-game progression without unnecessary complexity.

## Why?

This mod is made as my project for my thesis. If you have some feedback, please contact me: @marneyy_


## Translation

Mod is in two languages, English (US) and Czech.


## What did the mod add?

Tin Related
- Tin Ingot
- Raw Tin
- Tin Ore (normal and deepslate variant)
- Tin Booster (block that gives you haste I upon standing on it)
- Tin Nugget
- Tin Dust
- Raw Tin Block
- Tin Block

Copper Related
- Copper Nugget
- Copper Dust

Bronze Related
- Bronze Ingot
- Bronze Dust
- Bronze Block
- Bronze Chisel

Lignite Coal Related
- Ligntie Coal Ore (deepslate and normal version)
- Lignite Coal
- Lignite Coal Block
- Useable as fuel, 1600 and 16000 burn value respectively


## Compatibility
- Minecraft: 1.21.1  
- Fabric Loader: 0.16+  
- Fabric API: latest version recommended  


## Installation
1. Download the latest release from the Releases page.  
2. Install Fabric Loader.  
3. Place the `.jar` file into your `mods/` folder.  
4. Launch the game.
