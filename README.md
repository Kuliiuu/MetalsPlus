# MetalsPlus (Fabric 1.21.1)

MetalsPlus is a lightweight and expandable Fabric mod for Minecraft 1.21.1 that introduces new metals, materials, and later world generation features. The goal of the mod is to enhance mid-game progression without unnecessary complexity.

## Features and Planned Additions

### Tin (planned)
- New ore: Tin Ore  
- Raw Tin and related processing  
- Tin Ingot  
- Tin Tools (pickaxe, axe, shovel, sword, hoe)  
- Tin Armor (helmet, chestplate, leggings, boots)

### Bronze (planned)
- Bronze Ingot  
- Bronze Tools (pickaxe, axe, shovel, sword, hoe)
- Bronze Armor (helmet, chestplate, leggings, boots)

### Structures (planned)
- A new structure related to mining (name to be defined)

## Compatibility
- Minecraft: 1.21.1  
- Fabric Loader: 0.16+  
- Fabric API: latest version recommended  

## Installation
1. Download the latest release from the Releases page.  
2. Install Fabric Loader.  
3. Place the `.jar` file into your `mods/` folder.  
4. Launch the game.
